Politique de confidentialité

Dernière mise à jour : 10 novembre 2024

En résumé : Bonballon ne collecte aucune donnée personnelle. Toutes vos informations restent sur votre appareil. Nous respectons totalement votre vie privée.

1. Collecte de données

Bonballon est une application qui fonctionne entièrement en local sur votre appareil. Nous ne collectons, ne stockons et ne transmettons aucune donnée personnelle à des serveurs externes.

2. Stockage local

L'application utilise le stockage local (localStorage) de votre navigateur pour sauvegarder les informations suivantes uniquement sur votre appareil :

  • Tirages d'équipes : Les équipes que vous avez générées
  • Historique des matchs : L'historique de vos tirages précédents
  • Contraintes personnalisées : Vos préférences de joueurs ensemble/séparés et capitaines
  • Statistiques de compétition : Les résultats et classements du mode compétition
  • Historique des écarts-types : Les statistiques d'équilibrage de vos tirages

3. Partage de données

Aucune donnée n'est partagée avec des tiers, des serveurs externes ou des services d'analyse. Toutes vos informations restent strictement sur votre appareil.

4. Fonctionnalités de partage

L'application propose des fonctionnalités de partage optionnelles :

  • Partage WhatsApp : Lorsque vous utilisez cette fonction, seul le texte des équipes est partagé via l'application WhatsApp installée sur votre appareil
  • Export PDF : Les fichiers PDF sont générés localement et sauvegardés sur votre appareil
  • Export/Import JSON : Les fichiers de données sont créés et lus localement

5. Permissions requises

L'application peut demander les permissions suivantes :

  • Stockage : Pour sauvegarder les exports PDF et JSON sur votre appareil
  • Aucune autre permission n'est requise

6. Cookies et technologies de suivi

Bonballon n'utilise aucun cookie, aucun tracker publicitaire, ni aucune technologie de suivi. Nous ne collectons aucune donnée analytique.

7. Sécurité des données

Étant donné que toutes les données sont stockées localement sur votre appareil :

  • Vos données sont aussi sécurisées que votre appareil
  • Nous vous recommandons de protéger votre appareil avec un code PIN/mot de passe
  • Si vous désinstallez l'application, toutes les données locales seront supprimées

8. Droits des utilisateurs

Vous avez un contrôle total sur vos données :

  • Accès : Toutes vos données sont accessibles via l'application
  • Export : Vous pouvez exporter vos données à tout moment (format JSON)
  • Suppression : Vous pouvez supprimer vos données via les boutons "Effacer l'historique" ou "Réinitialiser" dans l'application
  • Portabilité : Vos données exportées peuvent être importées sur un autre appareil

9. Services tiers

L'application utilise les bibliothèques suivantes, toutes chargées depuis des CDN publics :

  • jQuery (bibliothèque JavaScript)
  • Bootstrap (framework CSS)
  • Font Awesome (icônes)
  • Chart.js (graphiques)
  • html2canvas et jsPDF (export PDF)

Ces bibliothèques sont utilisées uniquement pour le fonctionnement de l'application et ne collectent aucune donnée.

10. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Toute modification sera publiée sur cette page avec une nouvelle date de mise à jour.

11. Application pour enfants

Bonballon est adapté à tous les âges. L'application ne collecte aucune information personnelle, ce qui la rend conforme aux réglementations sur la protection des enfants en ligne (COPPA, RGPD).

12. Contact

Pour toute question concernant cette politique de confidentialité ou l'utilisation de vos données, vous pouvez nous contacter :
